# TurboChat3000
Realisation d'un TP de java permettant de creer un chat entre plusieurs clients

/!\ Le serveur est rêglé pour utiliser le serveur 64998, mais il est possible de le changer dans Serveur.java /!\
/!\ Bien penser a lancer le Serveur.java avant de demander aux clients de se connecter /!\

Petite interface permettant de discuter entre différents clients.

Possible de se connecter avec le nom de son choix.
Possible de consulter l'historique de messages.
Chaque client connecté à une couleur.
Possibilité de voir en temps réel qui est connecté au serveur.
